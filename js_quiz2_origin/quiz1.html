Javascript Quiz

Q1. Boolean(10 > 9) 코드가 반환하는 값은?

  1. true
  2. false
  3. NaN

(해설) 10은 9보다 크므로 참(true) 값이다.

Q2. 다음 중 결과가 다른 하나는?

  1. !true
  2. true && false
  3. true || false
  4. !!false && (3 > 5)

(해설) 1, 2, 4는 false 3번은 true값이다.

Q3. 다음 중 결과가 다른 하나는?

  1. 3 !== '3'
  2. (8 / '2') !== 4
  3. 0 === false
  4. ('1' + 2) === 12

(해설) ==, != 연산자는 두 피연산자의 타입이 다르더라도 값만 비교하지만
===, !== 연산자는 엄격한 동등 연산자로 피연산자의 값과 타입을 모두 비교한다.

Q4. 다음 중 if 문을 올바르게 작성한 것은 무엇인가?

  1. if (x == 5)
  2. if (x == 5):
  3. if (x == 5) then
  4. if x = 5

(해설) 조건문은 if (조건식)으로 작성한다.

Q5. 자바스크립트에서 "i가 7이 아닐 때"를 의미하는 코드로 올바른 것은?

  1. if (i <> 7)
  2. if (i !== 7)
  3. if (i =! 7) then
  4. if i != 7:

(해설) i가 7이 아닐 때이므로 동등 연산자 중에서 != 혹은 !== 연산자를 이용해야 하며, if (조건식) 형식으로 작성해주어야 한다.
